# Environmental damages exceed production value in global livestock systems

This repository contains the notebooks and file structure used to reproduce the welfare accounting workflow for livestock production. The analysis monetizes three environmental damage channels—enteric and manure greenhouse gas emissions, land-use carbon opportunity costs, and nitrogen pollution—and compares them with livestock production value under low, mid, and high social cost scenarios.

## Repository structure

```text
repo/
├── code/
│   01_SCC_enteric_manure_emissions_cost_country_year.ipynb
│   02_SCC_pasture_cropland_cost.ipynb
│   03_SCN_nitrogen_cost.ipynb
│   04_welfare_analysis_and_figures.ipynb
├── data/
│   ├── raw/
│   └── outputs/
└── figures/
```

## Workflow

Run the notebooks in this order:

1. **01_SCC_enteric_manure_emissions_cost_country_year.ipynb**  
   Computes enteric fermentation and manure management damages from FAOSTAT emissions using three social cost of carbon assumptions.

2. **02_SCC_pasture_cropland_cost.ipynb**  
   Computes livestock land opportunity costs using cropland and pasture area, feed allocation, recoverable carbon stock assumptions, and SCC scenarios.

3. **03_SCN_nitrogen_cost.ipynb**  
   Computes livestock-attributable nitrogen surplus and monetizes nitrogen damages using three social cost of nitrogen scenarios.

4. **04_welfare_analysis_and_figures.ipynb**  
   Integrates the damage components, calculates welfare balances, and produces the main manuscript figures.

## Data folders

- `data/raw/` contains original source data downloaded from FAOSTAT.
- `data/outputs/` contains intermediate and final processed files generated by the notebooks.
- `figures/` contains exported `.png` and `.pdf` figures for the manuscript.
